Var Let Const

var has a confusing scoping behavior and can lead to unexpected results. It's better to use let and const instead. If you need to reassign a variable, use let. If a variable's value doesn't need to change, use const. This improves code readability and helps prevent accidental reassignments.

2. What is the difference between find(), filter() method?

find() Method. filter() Method.
It return the first element in the array that satisfies the provided testing function. If no element matches, it returns undefined. It returns a new array containing all the elements that satisfy the provided testing function. If no elements match, it returns an empty array.
It is very useful when you need to locate a single element in an array. Useful when you need to extract multiple elements from an array based on certain conditions.
Ex:
const numbers = [1, 2, 3, 4, 5];
const foundNumber = numbers.find (num => num > 2);
console.log (foundNumber); // Output: 3		 Ex:
const numbers = [1, 2, 3, 4, 5];
const filteredNumbers = numbers.filter (num => num > 2);
console.log (filteredNumbers); // Output: [3, 4, 5]
